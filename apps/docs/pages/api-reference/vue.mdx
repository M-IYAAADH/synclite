# @nexsync/vue

Vue 3 composables for NexSync. All composables return reactive `Ref` values.

## `createNexSync(config)`

```typescript
function createNexSync(config: NexSyncConfig): Plugin
```

Create a Vue plugin. Install with `app.use()`.

```ts
app.use(createNexSync({ relay: 'wss://...', appId: 'my-app' }))
```

## `useNexSyncSetup(config)`

```typescript
function useNexSyncSetup(config: NexSyncConfig): NexSync
```

Set up NexSync inside a root component and provide it to descendants.
Call this instead of `createNexSync()` when you want NexSync scoped to a component subtree.

## `useNexSync()`

```typescript
function useNexSync(): NexSync
```

Access the NexSync instance from any descendant component.

## `useValue(key)`

```typescript
function useValue(key: string): Ref<Record<string, unknown> | null>
```

## `useQuery(prefix)`

```typescript
function useQuery(prefix: string): Ref<Record<string, Record<string, unknown>>>
```

## `useStatus()`

```typescript
function useStatus(): Ref<'connecting' | 'connected' | 'offline' | 'syncing'>
```
