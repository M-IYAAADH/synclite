# @nexsync/react

React 18+ hooks and provider for NexSync.

## `<NexSyncProvider>`

```typescript
type NexSyncProviderProps = NexSyncConfig & { children: React.ReactNode }
```

Wrap your app (or a subtree) to make NexSync available via hooks.

```tsx
<NexSyncProvider relay="wss://relay.example.com" appId="my-app">
  <App />
</NexSyncProvider>
```

Multiple providers are supported — each creates an isolated NexSync instance.

## `useNexSync()`

```typescript
function useNexSync(): NexSync
```

Access the raw `NexSync` instance for writes and subscriptions.

## `useValue(key)`

```typescript
function useValue(key: string): Record<string, unknown> | null
```

Subscribe to a single key. Returns `null` until a value is set.

## `useQuery(prefix)`

```typescript
function useQuery(prefix: string): Record<string, Record<string, unknown>>
```

Subscribe to all keys under a prefix. Returns an empty object initially.

## `useStatus()`

```typescript
function useStatus(): 'connecting' | 'connected' | 'offline' | 'syncing'
```

Track the current relay connection status.
