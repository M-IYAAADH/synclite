# NexSync

**Offline-first sync for any app.** Install one package, get instant local writes, automatic conflict resolution, and real-time sync — without thinking about network state.

```bash
npm install @nexsync/core
```

```typescript
import { NexSync } from '@nexsync/core'

const db = new NexSync({ relay: 'wss://relay.example.com' })
db.set('note:1', { title: 'Hello' })  // instant locally, syncs automatically
```

That's it. Works offline. Syncs on reconnect. No conflicts to resolve manually.

## Features

- **Instant local writes** — `db.set()` writes to IndexedDB in < 5ms. No spinners.
- **Automatic sync** — Changes flow to all connected clients via a WebSocket relay.
- **Offline queue** — Writes queue locally when offline and flush on reconnect.
- **LWW conflict resolution** — Last-Write-Wins with Lamport timestamps. Delete always wins.
- **Self-hostable relay** — One Docker command. Your data stays on your infra.
- **Framework adapters** — React, Vue 3, React Native. Vanilla JS works too.
